\subsection{Spectral Estimation}

For each ROI, the direction-averaged signal decay (@comment stephan: geometric mean of signals measured along three orthogonal diffusion encoding directions) was decomposed into eight diffusivity components using both MAP and fully Bayesian estimation as described in Theory.

\subsubsection{MAP Estimation}

MAP spectra were computed via Eq.~\ref{eq:map} with ridge strength $\lambda = 0.1$.
The resulting eight-component spectra were normalized to sum to one (fractional representation).
Computation was instantaneous ($< 1\;\text{s}$ for all 149 ROIs).

\subsubsection{Bayesian Estimation (NUTS)}

The posterior (Eqs.~\ref{eq:prior_R}--\ref{eq:likelihood}) was sampled independently for each of the 149 ROIs using NUTS via PyMC, with 2000 post-tuning draws, 200 tuning steps, 4 independent chains, and target acceptance rate 0.95.
Convergence was assessed using the split-$\hat{R}$ statistic \citep{vehtari2021rank}; all 149 ROIs converged ($\hat{R} < 1.05$).
Posterior means of the spectral fractions (normalized to sum to one) were used as NUTS point estimates; posterior standard deviations quantify per-component uncertainty.

\subsection{ADC Computation}

Apparent diffusion coefficients were computed by monoexponential least-squares fitting of $\log S(b) = \log S_0 - b \cdot \text{ADC}$ to the raw signal at b-values $\leq 1000\;\text{s/mm}^2$ (five measurements per ROI: $b = 0, 250, 500, 750, 1000\;\text{s/mm}^2$).

\subsection{Classification}

Tumor detection and Gleason grade classification were evaluated using $L_2$-regularized logistic regression with leave-one-out cross-validation (LOOCV).
Three classification tasks were defined: (1)~tumor versus normal in PZ ($n = 81$), (2)~tumor versus normal in TZ ($n = 68$), and (3)~high-grade (GGG~$\geq 3$) versus low-grade (GGG~1--2) among tumor ROIs with available GGG labels ($n = 29$).

For each task, three feature representations were compared: (a)~ADC alone (single feature), (b)~all eight MAP spectral fractions, and (c)~all eight NUTS posterior mean fractions.
For single-feature models (ADC, individual components), the raw feature value was used directly as the classification score without training a classifier.
For multi-feature models, standardized features were passed through logistic regression with regularization $C = 1.0$ (inverse regularization strength), with separate standardization and model fitting in each LOOCV fold to prevent data leakage.
Performance was measured by the area under the receiver operating characteristic curve (AUC).
Sensitivity analyses at $C = 0.1$ and $C = 10.0$ confirmed that results were robust to regularization strength.

\subsection{ADC Sensitivity Analysis}

The ADC sensitivity vector (Eq.~\ref{eq:sensitivity}) was computed numerically at the average tumor and average normal spectra for each zone, using finite differences with step size $\epsilon = 10^{-4}$.
Logistic regression coefficients were obtained by fitting the full model (all eight features) to the entire dataset.
The Pearson and Spearman rank correlations between the sensitivity vector and the LR coefficient vector (both eight-dimensional) were computed to quantify alignment.
